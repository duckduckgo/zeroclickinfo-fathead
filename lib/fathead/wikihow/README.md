wikiHow Fathead
===

The wikiHow fathead.
This project requires Python 3.3+

```
$ cd lib/fathead/wikihow
$ pyvenv .
$ source bin/activate
$ pip install --upgrade pip
$ pip install -r requirements.txt
$ ./fetch.sh
$ ./parse.sh
```

Contact [PJ Hampton](https://github.com/pjhampton) if you need any help.
