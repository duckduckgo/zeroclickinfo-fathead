# PEP Fathead

This Fathead parses the various PEPs (Python Performance Proposals) from the PSFs website.

### IA Page

https://duck.co/ia/view/pep

### Dependancies

  - Python 3.5.2
  - BeautifulSoup 4 (bs4)

### Running

```
$ bash fetch.sh # gets the seed document
$ bash parse.sh # creates the output file
```
