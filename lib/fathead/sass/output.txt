Variable Arguments	A										<section class="prog__container"><p>Sometimes it makes sense for a mixin or function to take an unknown number of arguments. :expanded	A										<section class="prog__container"><p>Expanded is a more typical human-made CSS style, with each property and rule taking up one line. :compact	A										<section class="prog__container"><p>Compact style takes up less space than Nested or Expanded. It also draws the focus more to the selectors than to their properties. For example:</p><pre class="code ruby"><code class="ruby">#main { color: #fff; background-color: #000; }\n#main p { width: 10em; }\n\n.huge { font-size: 10em; font-weight: bold; text-decoration: underline; }</code></pre></section>	http://sass-lang.com/documentation/file.SASS_REFERENCE.html#_15
