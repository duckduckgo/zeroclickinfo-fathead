    <section class="prog__container">
        <p>Creates an array of elements split into groups the length of size. If array can't be split evenly, the final chunk will be the remaining elements.</p>
        <pre><code>_.chunk(array, [size=1])</code></pre>
        <span class="prog__sub">Arguments</span>
        <pre><ol><li><code>array</code> <em>(Array)</em>: The array to process.</li><li><code>[size=1]</code> <em>(number)</em>: The length of each chunk</li></ol></pre>
        <span class="prog__sub">Returns</span>
        <pre><p><em>(Array)</em>: Returns the new array of chunks.</p></pre>
